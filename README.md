This repository contains implementations of **Decision Tree algorithms** for both **Classification and Regression** using Python and Jupyter Notebooks.

It focuses on building a strong understanding of **tree-based models**, which are widely used in real-world Machine Learning systems.

---

## 📌 Project Overview

This project includes:

### 🔹 Decision Tree Classifier
- Used for **classification problems**
- Splits data based on feature conditions
- Easy to interpret and visualize

---

### 🔹 Decision Tree Regressor
- Used for **predicting continuous values**
- Captures **non-linear relationships**
- Works well with complex datasets

---

## 🧠 Key Concepts Covered

- Supervised Learning  
- Decision Tree Algorithm  
- Entropy & Information Gain  
- Gini Index  
- Overfitting & Underfitting  
- Tree Depth & Pruning
